# gcp-lz-products

Repo that provides the basic aggrgated modules that can be used to create resources on GCP. It is used by project and foundation code to spin of various key resources like below:

```code
├── cloud-dns
│   └── private
├── devops
├── gke
│   ├── gke-autopilot
│   ├── gke-standard
│   └── gke-vm
├── management
│   └── src
│       └── guardrails
├── network
├── platform-tools
├── shared-vpc
└── vm-windows
```

`tree -d -I docs`
